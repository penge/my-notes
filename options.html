Keyboard Shortcuts

Windows or Linux

Ctrl + Shift + 1 Open page 1
Ctrl + Shift + 2 Open page 2
Ctrl + Shift + 3 Open page 3
Ctrl + Shift + F Toggle Focus mode

Mac

Option + 1 Open page 1
Option + 2 Open page 2
Option + 3 Open page 3
Option + F Toggle Focus mode

Options

Focus mode hides bottom panel (Options link and page number) to give you more editing space.
To change this option back, visit Options (right-click on My Notes icon in browser's toolbar), or press "Toggle Focus mode" keyboard shortcut.
